🃏 WordCard
A Django-based English vocabulary learning app powered by the SM-2 spaced repetition algorithm and AI-generated sentence practice via the Groq API.

✨ Features

Spaced Repetition (SM-2) — Cards are scheduled based on how well you remember them. The better you know a word, the less often it appears.
AI Sentence Practice — After reviewing a word, you can generate contextual example sentences using the Groq API (LLM-powered).
Flashcard System — Clean card-based UI for focused vocabulary sessions.
Progress Tracking — See which words you're mastering and which need more work.
Persistent Storage — All words and review history saved to a MySQL database.
